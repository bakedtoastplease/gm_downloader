# 🛠️ gm_downloader 🚀

**gm_downloader** — это мощная и простая консольная утилита для автоматической загрузки и установки аддонов из Steam Workshop прямо в пиратскую версию Garry's Mod. Больше никакой ручной распаковки!

---

## ✨ Особенности
* **Автоматизация:** Скачивает и распаковывает аддоны одной командой по их ID.
* **Поддержка баз и Pill Packs:** Успешно устанавливает даже сложные скриптовые модификации (Parakeets Pill Pack и др.).
* **Универсальность:** Благодаря файлу конфигурации `set.ini` можно настроить пути для любой сборки игры.
* **Чистая установка:** Моды попадают сразу в папку `addons/` в готовом для игры виде.

---

## ⚙️ Требования
1.  **SteamCMD:** Официальная консоль Steam (должна быть в папке с программой).
2.  **Права Администратора:** Программу необходимо запускать от имени администратора для записи файлов в системные папки.
3.  **Windows 10/11** (Тестировалось на Windows 10 LTSC).

---

## 🚀 Инструкция по установке

1.  Скачайте репозиторий или скомпилированный `.exe`.
2.  Поместите `steam_cmd.exe` в корневую папку программы.
3.  Отредактируйте файл `set.ini`, указав путь к вашей папке аддонов:
    ```ini
    [Settings]
    addonspath = C:\Games\Garrys Mod\garrysmod\addons
    ```
4.  Запустите `gm_downloader.exe` **от имени администратора**.

---

## 🎮 Как пользоваться

1.  Найдите нужный аддон в [Мастерской Steam](https://steamcommunity.com/app/4000/workshop/).
2.  Скопируйте **ID** из адресной строки (цифры в конце ссылки `?id=XXXXXXXXX`).
3.  Введите этот ID в консоль программы.
4.  Дождитесь надписи `Успех!` и заходите в игру!

## 🛠️ Разработка
